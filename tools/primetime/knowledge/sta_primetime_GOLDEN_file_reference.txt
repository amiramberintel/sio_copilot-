clock_collateral/
    Clock definitions and constraints for ALL corners (36 subdirs).
    Each corner has: *_clocks.tcl, *_clock_params.tcl, clock_uncertainty.tcl
    Corners include: func.max_*, func.min_*, fresh.*, noise.*

constraints_mapping/
    Mapping files that translate constraint names between tools.

dft_collateral/
    DFT (Design For Test) related files - scan chain definitions,
    ATPG constraints, BIST collateral.

fe_collateral/
    Front-End collateral from RTL:
    - par_msid.f          : Verilog filelist
    - par_msid.v.gz       : RTL Verilog source
    - par_msid.upf        : Power intent (UPF format)
    - par_msid.hip_tags.xml : HIP (Hierarchical IP) tags
    - ms_units/           : Memory/stub unit definitions
    - par_msid_rtl_snapshot/ : RTL snapshot for FEV

fscript/
    Flow scripts used for the STA setup and analysis.

fv/
    Formal Verification related files and reports.

genus_dft/
    Cadence Genus DFT synthesis outputs and reports.

hip_data/
    HIP (Hierarchical IP) data - timing models and abstracts for
    hard macros used in the design.

logs/
    Log files from the STA runs and archiving process.

par_msid.db/
    Cadence Innovus database - complete P&R database including:
    - par_msid.fp.gz      : Floorplan
    - par_msid.place.gz   : Placement
    - par_msid.route.gz   : Routing (868MB)
    - libs/               : Library data
    - mmmc/               : Multi-Mode Multi-Corner setup
    - extraction/         : Parasitic extraction data

par_msid.gpd/
    Gate-level Power Data - used for power analysis.

par_msid.pt_session.func.max_nom.T_85.typical/

pivot/
    Pivot table data for timing analysis summaries.

power_collateral/
    Power intent files - UPF, power domains, level shifters.

reports/
    Additional reports directory (contains star_pv extraction reports).

sio_timing_collateral/
    SerDes/IO timing collateral - constraints for high-speed interfaces.

td_collateral/
    Top-Down collateral for integration:
    - icc2/      : Synopsys IC Compiler 2 format

timing_collateral/
