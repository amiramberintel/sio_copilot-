  -----------------------------------------------------------------------
  FIX A: TIP TOPOLOGY PLANNING (13-16 ps gain)
  -----------------------------------------------------------------------
  Request new TIP topology file: icore.ilgbpqidm107h.tp

  Recommended TIP constraints (based on ifhitextsnoopm105h reference):

    a) Pin bbox constraints:
       - par_msid pin: bbox_percentage optimized for lower Y
         (move port closer to source logic at Y ~ 124K)
       - par_fe pin: bbox_percentage optimized toward endpoint at Y ~ 266K

    b) Layer assignment:
       - shape_layers: M17 100%, M18 100%
       - Upper metals have ~2x lower sheet resistance

    c) Repeater planning:
       - 3-5 repeaters with 1.5x sizing multiplier
       - Evenly spaced along topology edge
       - repeater_group_constraints with M17/M18 layer cutting

    d) Net estimation rule:
       - Create bundle for all 6 bits [0:5]
       - Single topology plan covering entire bus

  Expected improvements:
    - Port Y optimization:     saves ~5-8 ps wire delay
    - M17/M18 layer routing:   saves ~5-8 ps (lower RC)
    - Optimized repeater plan: saves ~3-5 ps
    - par_msid TIP chain: from 31.2 ps → ~15-18 ps

    e) Half-shield track blocking (half_shield_1w on M17/M18):
       - Blocks adjacent tracks on both M17 and M18 to shield TIP wires
       - Reduces crosstalk coupling capacitance → lower wire delay
       - Uses pre.tip_implementation.tcl definitions:
           M17_half_shield_1w = "{1.140 3.420} {0.200 0.200}"
           M18_half_shield_1w = "{1.235 3.705} {0.450 0.450}"
       - ivar entries to add to pre.tip_implementation.tcl:
           set ivar(tip_block_track_for_pin_placment,M17,ilgbpqidm107h_tp) "$M17_half_shield_1w 1"
           set ivar(tip_block_track_for_pin_placment,M18,ilgbpqidm107h_tp) "$M18_half_shield_1w 1"
           set ivar(tip_block_track_for_route,M17,ilgbpqidm107h_tp) "$M17_half_shield_1w 1"
           set ivar(tip_block_track_for_route,M18,ilgbpqidm107h_tp) "$M18_half_shield_1w 1"
       - Estimated additional gain: ~3-5 ps (reduced coupling)
