1. POWER RAILS DEFINED
================================================================================

  Rail          Description                        Voltage Range
  -----------   --------------------------------   ----------------
  VSS           Ground                             Always 0V
  VCCCORE       Core logic supply                  0.540V - 1.271V
  VCCRING       Ring/IO logic supply               0.540V - 1.271V
  VCCST         Always-on / Standby supply         0.550V - 1.271V
  VCCANA_EHV    Analog high voltage                Fixed 1.800V

Rail Grouping - treats these rails as having same voltage:
  VCCSA     -> grouped with VCCST
  VCCTM     -> grouped with VCCCORE
  VCCMLCSSA -> grouped with VCCCORE

================================================================================
KEY TAKEAWAYS
================================================================================

1. pvt.tcl is the FOUNDATION for MCMM (Multi-Corner Multi-Mode) analysis

2. Ensures timing closure across ALL operating conditions:
   - Multiple voltages (0.54V to 1.27V)
   - Multiple temperatures (-40C to 125C)
   - Multiple process corners (slow, typical, fast)

3. Different EDA tasks use different subsets of scenarios:
   - Synthesis: minimal (speed)
   - P&R: moderate (balance)
   - Signoff: comprehensive (accuracy)

4. Setup vs Hold:
   - Setup: func.max_* scenarios (slow paths, aged transistors)
   - Hold: fresh.min_* scenarios (fast paths, fresh transistors)
