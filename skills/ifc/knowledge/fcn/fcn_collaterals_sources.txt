=== WHAT TO DO WHEN YOU GET EACH EMAIL ===

  Basel's RTL email:
    -> Update ivar(fcn,fe_model) in fcn_user_vars.tcl
    -> This triggers a new FCN run

  Maxim's HIP email:
    -> Update ivar(fcn,hip_wa) in fcn_user_vars.tcl
    -> If ONLY HIP changed, can use HIP writer (no full run needed)

  Haya's clock tag email:
    -> Update ivar(fcn,clock_tag) in fcn_user_vars.tcl
    -> IMPORTANT: Use the BU_FCN tag, never the prop tag
    -> Clock tag changes can affect FDR counts (ssh_ijtag/ijtag_tck)

  Basel's contour pilot email:
    -> Run the contour pilot flow (separate from main FCN)
    -> Use contour pilot GUI or manual steps
    -> Report errors back to Basel
