=== SEC 9: GOTCHAS ===

  CRITICAL parsing traps (avoid repeating):

  1. IFC parser: line.startswith('-') skips ALL negative WNS!
     FIX: use line.startswith('-WNS') to skip only the header

  2. XML slack regex: slack="(-?\d+) matches normalized_slack BEFORE slack!
     FIX: use ' slack="(-?\d+)' with a LEADING SPACE

  3. Clock names: NOT all clocks have mclk_ prefix.
     Hold corners: idvfreqOut, idvfreqOut_div2/4/8/16, dcmcbbclk,
     ijtag_tck, scanclk_div, etc. Always show FULL clock names.

  IFC report direction: first port's partition = source,
  second unique partition = destination.

  Match key: strip icore0/par_<name>/, strip [*]{}, lowercase,
  remove _op_N, _feedthru*.

  Hierarchy: always use icore0/par_<name>/signal (not relative).
  icore0 = primary core, skip icore1.
