================================================================================
                         WHAT THIS REPORT MEANS
================================================================================

Unconstrained endpoints are OUTPUT PORTS where the timing path TO them has
no max_delay constraint (set_output_delay). This means:

  - PT cannot check if data arrives at this output within timing budget
  - The path is NOT verified for setup time at the receiving partition
  - Could cause timing failures at chip level if not constrained elsewhere
